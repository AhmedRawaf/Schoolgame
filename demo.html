اختبار تفاعلي لتعلم العناصر الكيميائية

🎯 هذا ملف تجريبي (Demo)

هذا الملف يعرض واجهة التطبيق فقط. للتطبيق الكامل، تحتاج لتشغيل Flask

مميزات الاختبار:

  • أسئلة متنوعة عن العناصر الكيميائية
  • ثلاثة مستويات صعوبة
  • مؤقت لكل سؤال
  • مساعدة 50:50 (مرة واحدة)
  • حفظ النتائج في ملفات CSV
  • مراجعة مفصلة للإجابات

🚀 كيفية تشغيل التطبيق الكامل:

الخطوة 1: تأكد من تثبيت Python 3.7+

الخطوة 2: افتح موجه الأوامر في مجلد التطبيق

الخطوة 3: قم بتشغيل الأوامر التالية:

pip install -r requirements.txt
python app.py

الخطوة 4: افتح المتصفح على http://localhost:5000

أو استخدم ملف setup.bat: انقر مرتين على setup.bat للتثبيت التلقائي
